# Product images

Replace each placeholder with your real product image and update the matching
`image` path in `lib/products.ts` if the extension changes.

Recommended format: WebP or AVIF, 1200 × 1400 px, portrait 6:7 ratio, under 250 KB.

Stable filenames:

- `argan-oil-shampoo.webp`
- `anti-hair-loss-shampoo.webp`
- `seamless-leggings.webp`
- `slim-body-shapewear.webp`
- `womens-sportswear.webp`

The current `.svg` files are lightweight placeholders and can be deleted after
the real images are connected.
